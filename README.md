# 📊 Interactive MIS Sales Dashboard

## 🔍 Project Overview
This is an end-to-end *Excel MIS Dashboard* built to track regional sales performance. It automatically finds the top/bottom sales reps and calculates how close they are to meeting their targets. 

It features a clean visual layout and uses *VBA Macros* to let users switch between different dashboard views with the click of a button.

---

## 🛠️ Key Features & Technical Skills

* *Data Workflow:* Followed a clear 4-step process: Understanding Data ➡️ Building Logic ➡️ Applying Formulas ➡️ Creating Reports.
* *Top/Bottom 5 Filters:* Used advanced Pivot Table settings to instantly isolate the top 5 and bottom 5 sales executives.
* *Target Tracking:* Calculated exact target hit percentages and mapped out performance gaps ("Away from Target %").
* *Connected Slicers:* Added interactive region buttons (Mumbai, Pune, Chennai, etc.) that update all charts at the same time.
* *Custom Styling:* Redesigned the default Excel slicers (changing colors for active and hovered items) for a professional, clean look.
* *VBA Macro Automation:* Wrote If...Then...Else VBA code linked to shape buttons to dynamically change report connections and switch dashboard views seamlessly.

---

## 📈 Visualizations Included
The dashboard is designed for full-screen executive viewing (Alt + V + U) with gridlines and headers hidden:
* *Bar Chart:* Shows the top sales executives with clear data labels.
* *Pie Chart:* Shows the percentage of targets successfully hit.
* *Line Chart:* Tracks trends of how far low performers missed their benchmarks.
* *Summary Tables:* Displays sorted data for deep-dive reviews.

## 🚀 How to Interact with the Live File

To experience the fully interactive elements and custom macro navigation buttons locally, please follow these quick steps to bypass Microsoft's default internet security blocks:

1. *Download* the End to End Project using Excel practice.xlsm file from this repository.
2. Before opening it, locate the file in your downloads folder, *right-click* it, and select *Properties*.
3. At the very bottom of the *General* tab, look for the *Security* section and check the box that says *Unblock*. 
4. Click *Apply* and then *OK*.
5. Open the file and click *Enable Content* or *Enable Macros* at the top bar.

Note: The interactive features require a desktop version of Microsoft Excel (2019 or newer recommended).
